# DiscussionTopics

Questions / Topics to use in discussion.

## Format

* ~30-60 minutes (post production edit is usually half the recorded duration)
* Google Hangout on Air
* Timeline
    * Introduction
    * General topics
    * Specific topics
    * Free style (if time allows)
